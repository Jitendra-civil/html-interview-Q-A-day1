1. DIV AND SPAN:

DIV:

  • In html div defines divison or a section, it is a block- level element.
  • div always uses full width available stretches as far as it can from left to right.
  • div always start with a new line , and the browser automatically add some space (margin) before and after the element
  • In css we can style the element in div tag ex:

    • div and span

SPAN:

  • In html span is a inline-element.
  • span tag is used to markup a part of a text, it only takes up much width as necessary
  • It doesn't start with a new line
  • In css we can style the element in span tag ex:

    • Hello world

2. CLASS AND IDS:

  • In html, the only difference between class and id that, id is unique and each element can have only one ID, whereas class are not unique they can be used multiple times.
  • ID: each page can have only one element with particular id
  • Class: we can use multiple classes on the same element.
  • In css, div is denoted by "#", class as "."

3. SELF CLOSING TAGS:

  • In HTML,self closing tags means in the name itself denotes that do not need closing tag to end.
  • It means no separate tag is used to terminate the tag.
  • ex: hr,br,img,input etc.

5. BLOCK AND INLINE ELEMENTS:

  • A block-level element always starts on a new line, and the browsers automatically add some space (a margin) before and after the element.
  • A block-level element always takes up the full width available (stretches out to the left and right as far as it can).
  • EX:div,p,form,hr,pre, etc...

  • An inline element does not start on a new line.
  • An inline element only takes up as much width as necessary.
  • ex: span, img, a

6. EM AND STRONG TAGS:

  • Em and strong tags are alternatives of b and i tag, we should prefer using em and strong
  • instead of b and i and for ranking of the elements
  • strong is used to show the importance of the text. and em is used to show the text in different
    manner, just to change the voice of tone.
  • ex: strong-Hello world, em -hello world

7. ATTRIBUTES:

  • Attributes provide additional information about elements
  • Attributes are always specified in the start tag
